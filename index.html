Instructions

  • This is a game where we test your memory.
  • In this game, a color will be chosen at random and shown to you.
  • After the color has been chosen, you need to click the colors in the sequence that they have been chosen by the game.
  • Keep in mind that once you pass level 1, only the new selected color will be shown to you, but you need to click the colors in the sequence with which it started.
  • Eg. If the game selects the color red in the first level, and then it selects the color green in the second level, you will need to click red first, followed by green. Similarly, if in the next level, the game selects color red again, you need to click in the sequence red - green - red.
  • Keep going as long as you can remember the sequence.
  • As soon as you press the wrong button/color, the game is over.
